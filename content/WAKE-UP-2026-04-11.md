**b. Blur-up lazy loader** — Use `<img class="pl-lazy" data-src="big.jpg" src="tiny.jpg">` and JS swaps src when the image enters viewport. IntersectionObserver with 200px rootMargin so images preload before they're visible. Falls back to immediate load on old browsers. Prevents CLS with a background placeholder color.

**c. WebP/AVIF picture wrapper** — Use `<img class="pl-modern" src="photo.jpg">` and JS wraps it in `<picture><source type="image/avif"><source type="image/webp"><img jpg fallback></picture>`. You just need to pre-generate `.avif` and `.webp` versions next to each `.jpg` in `/plants/`:
```bash
# Install once: brew install webp libavif
# Then in /plants/:
for f in *.jpg; do cwebp "$f" -o "${f%.jpg}.webp" -q 82; done
for f in *.jpg; do avifenc "$f" "${f%.jpg}.avif"; done
```
AVIF files are ~50% smaller than JPG, WebP ~30% smaller. Huge for mobile on 3G.

**d. HE ↔ EN language toggle** — A floating pill button in the top-left (top-right in RTL) with the label "English" / "עברית". Dispatches a `pl:langchange` event so other modules can react. Tested live in Chrome — toggle works bidirectionally, dir attribute flips correctly, button label updates correctly.
